## Rodando

```bash
pip install kivy pillow
python Ponyflix.py
```

Coloca o `episodios.json` e a pasta `assets/` (com `t1.png` até `t9.png`) na mesma pasta.

## Gerando APK

Use o Buildozer no linux, `buildozer.spec` incluso.

## Atualizando episódios


Só seguir o Json e adicionar na mesma logica.

Precisa do Chrome instalado.
